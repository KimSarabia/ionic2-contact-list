# IONIC 2 FRIEND LIST APP
Friend/Contact List
[DEMO](https://still-tundra-27579.herokuapp.com/)

## Installation
Download or git clone
* `npm install`
* `ionic serve`
* `nodemon server.js`

###Note: Persistence on localhost:8080 NOT localhost:8100

## Usage
* Add Friend
* Update Friend Info
* Delete Friend
* View All Friends
